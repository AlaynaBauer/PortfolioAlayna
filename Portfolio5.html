UX Testing Report

As a class we worked to provide an analysis of the CLASS website. We needed to analyze how users were using the current page and how they would navigate to a page that doesn’t exist yet so that we can utilize the results in redesigning the website and in designing the new page.

Process

We tested using 3 users to evaluate what paths users would use to navigate to a certain page on the CLASS website that details the research currently being done by professors at the college (a page that doesn’t exist yet) and were tasked to write a report on the results.

Outcome

The result is this report detailing the results of 3 tests by users which resulted in finding that the most common paths and search terms used by users to imform how the page should be structured and placed in the future.
